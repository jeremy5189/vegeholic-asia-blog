[小異女孩日記] ✨敏弱肌也能大口吃的甜食們✨

2016-10-01 Leslie 發表迴響

今天要把小異女孩的甜食挑選小撇步、3分鐘私房甜點大公開!?

其實Leslie小編從小就很愛吃甜的，蛋糕餅乾都是我的最愛，對巧克力更是一點抵抗力都沒有(難怪小時候常蛀牙?)

接觸到健康飲食觀念後，才發現高GI值(高升糖指數)的甜點容易讓血糖不穩定，進而刺激脂肪生成發炎反應。

有看上一篇【小異女孩日記】有效對抗異位性皮膚炎的日常照護方法?就會知道，除了第二型糖尿病外，導致異位性皮膚炎的元凶正是慢性發炎!

而一般市面上充滿精緻糖、高密度動物性脂肪(鮮奶油、無鹽奶油等)的甜點正是會讓血糖有如雲霄飛車的罪魁禍首。

那…為了避免發炎，敏感體質的人就一丁點甜的都不能吃嗎??

好消息是，只要盡量選擇用?非精緻糖及?好脂肪製作的甜品，敏弱肌一樣能享受吃甜食的樂趣!

但身為大學生的我也了解，大學附近健康的外食正餐選擇已經很少，要找到健康的甜點更是難上加難。就算有，常常也是對學生來說較不友善的價格。

荷包扁扁?又沒時間時，沒有沒有能滿足sweet tooth不導致發炎，又不用花時間訂購或大老遠跑到有機甜品店買的替代方案呢?

有的!???

Leslie小編在這邊要跟大家分享敏弱肌不論 #嘴饞時  #外食  #在家裡有閒時 都能放心享受甜品的小撇步。

 

✨嘴饞時✨

下午三四點時，午餐已經消化得差不多但又還沒到晚餐時間，肚子總是會有點小餓，這時候就很容易開始嘴饞想吃甜的~

會嘴饞，其實是因為大腦接收到血糖開始下降的訊息，告訴身體要補充營養囉! 這也是為什麼正餐吃太少的人容易一直想吃零食的原因 ?

這時候最壞的情快就是為了圖方便，隨便去便利商店買巧克力條或喝杯全糖的手搖飲料?

更好的選擇有?

1.香蕉

香蕉是最方便的甜點了! 甜度高能滿足想吃甜食的慾望外、取得便利性高、營養程度的cp值也很高?

一根香蕉裡除了含有優質碳水化合物能迅速補充能量外，也因為含有血清素，吃了能讓人心情變好，幫助正向思考提高學習、工作效率外，香蕉裡更有豐富的鉀能改善久坐久站造成的小腿浮腫唷!((女孩們快筆記?

( photo: Health Anti-aging Wellness )

雖然一根要十幾塊不算划算，但比起一條25元以上的巧克力棒或麵包還是便宜得多，營養cp值又高~真的是理想的afternoon pick me up snack!

 

2.椰棗堅果

椰棗吃起來根本就跟固化焦糖一模一樣! 在Costco就有賣而且價格合理，一大包可以吃超久才300多塊?

椰棗是一種來自阿拉波地區的果實，原本是長在樹上的?

( photo: C@MARADERIE )

椰棗是一種甜度非常高的果實，成分主要為單純的果糖。

那…都是果糖不會讓血糖飆高嗎?? 因為椰棗除了果糖外也含有豐富的纖維質和維生素，所以算是低GI食品，近幾年台灣也有漸漸開始流行的趨勢，甚至有糖尿病患者的最佳代糖稱號唷!

我最喜歡的吃法就是把椰棗撥開來，裡面放堅果(乍看有點像蜜紅棗包糯米中式點心心太軟?

像圖片中夾上喜歡的堅果醬或裹上巧克力醬吃感覺也很美味!?

( photo: betterwithcake.com )

用保鮮袋裝好，雖身放幾棵椰棗夾心在包包裡，就是方便又優質的甜點了!

平常開封後的椰棗只要冰冰箱裡就好，像果乾一樣不容易壞。

 

3. 果乾隨手包

( photo: lovely package )

 

期中考、或工作忙碌時，如果真得很忙、沒時間準備的時候，也可以去便利商店或超市買個果乾隨手包解解饞，但盡量挑選不含防腐劑、香料和色素的比較好。

 

✨外食選擇✨

肚子比較餓，吃個香蕉幾顆椰棗根本沒辦法果腹、又想吃甜食的時候， 該怎麼辦呢? 啊有時候就是沒去逛Costco?，哪來的椰棗啊?

其實外食也是有一些比較健康的甜品選項的?

 

1.豆花

如果本身沒對大豆過敏的話，豆花真的是零脂肪不容易發胖的健康甜品唷!

只要記得兩個選料小訣竅: 低升糖、看得見食物原本樣子的配料，例如: 花豆、紅豆綠豆、花生、麥片都是不錯的選擇。

( photo: 立達豆花 )

如果不確定店家是否為手工自製，粉圓、粉條這類容易有添加物的配料還是盡量少吃為妙~

另外可以注意一下店家是否有使用非基改黃豆的證明。現在不少有良心的店家都有標示使用原料檢驗證明，外食也別忽略了自己吃好食的權利☝

 

2. 日式蕨餅

市面上蕨餅多為用甘藷粉或木薯粉或製成的日式甜品，較道地的日式蕨餅則是用曬乾的蕨菜根磨成的粉加上糖和水製作而成。

蕨餅原料簡單，比起吃含大量氫化油和色素的餅乾外，是較好的甜食選擇?

( photo:  the-humble-hostess.blogspot )

近幾年從日本來的抹茶主題下午茶?一家接著一家開，跟朋友們或姊妹們聚會時，也不妨選擇成分簡單，吃了對身體比較沒有負擔的蕨餅~

搭配黃豆粉或抹茶粉的蕨餅QQ的好好吃☺

更棒的是，蕨餅不是用魚膠或吉利丁為原料，而是用植物性粉為原料，因此也是對麩質過敏者友善的甜品喔!

 

3. 裸食旦糕

可喜可賀! 近幾年台北很棒的植物性餐廳越開越多!像Plants、Oohchacha、綠帶純植物烘焙、Kayakayacafe、裸食私廚等餐廳都有提供用非精緻糖、椰子油、和堅果做的裸食旦糕。

( photo: 兩光主婦Eva的走跳日誌 )

這些店家提供的裸食旦糕製作原料不含奶蛋，都是全植物性、無麩質的，都是對過敏的朋友們非常友善的甜點。

試過就知道! 吃用廚師的愛心、而不是其他動物的傷心做的甜點，遠比用充滿添加物、動物性脂肪做的甜點更讓身體感到舒服。

有時間的話不妨邀請朋友一起到這些店家，一邊享受愜意的午後，一邊享用充滿營養又零殘忍的好吃旦糕吧!

 

 

✨在家有閒的時候✨

最後要教大家在家就能簡單做的甜點 ?三分鐘焦糖香蕉佐可可醬?

?材料: 香蕉一根、椰子油、生可可粉、椰子糖

?焦糖香蕉作法:

1. 開小火預熱平底不沾鍋，香蕉去皮切對半

2. 平底鍋塗上少許椰子油，香蕉切面朝下用中火煎

4. 煎到表面焦糖化後就可以起鍋擺盤

?可可醬作法: 將些許可可粉、椰子糖和和椰子油拌均勻、調到自己喜歡的稠度後，就可以淋在香蕉上。

最後加上自己喜歡的toppings像生可可脆粒、或granola增加一點口感，暖心的秋日甜點就完成了!

?小訣竅: 香蕉下鍋時火要開大，讓香蕉切面快速焦糖化。但切記不要煎太久讓一面都熟了，香蕉裡的酵素才不會被高溫破壞。

 

 

 

可能有人會認為，只是想吃個甜的搞得那麼複雜幹嘛?

其實會開始想跟大家分享這些小撇步希望傳達的想法是

不管是異位性皮膚炎患者、或對麩質過敏、甚至天生就有第一型糖尿病的人，也有能放心享受美食的權利。

只要多做一點小準備和注意幾個選擇要點，也能在吃零食和外食的同時，不用擔心會對身體造成不良影響地放心享受食物。

畢竟，能以放鬆的心情、好好享受眼前的好食，才是吃什麼都美味的秘訣啊!?
